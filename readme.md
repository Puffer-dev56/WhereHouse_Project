# ระบบจัดการคลังสินค้า (WMS Prototype)

ระบบต้นแบบ (prototype) สำหรับจัดการสินค้าคงคลัง พัฒนาด้วย HTML/CSS/JavaScript ล้วน ไม่ต้องติดตั้งหรือรัน server ใดๆ

---

## วิธีใช้งาน

เปิดไฟล์ `index.html` ในเว็บเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไรเพิ่ม

```
index.html   ← เปิดไฟล์นี้ในเบราว์เซอร์
README.md
```

---

## ฟีเจอร์

### เพิ่มสินค้าเข้าคลัง
- พิมพ์ชื่อสินค้าในช่อง "ชื่อสินค้า" ระบบจะแนะนำชื่อสินค้าที่มีอยู่แล้ว (datalist autocomplete)
- ถ้าชื่อตรงกับสินค้าที่มีอยู่ ระบบจะ **บวกจำนวนเพิ่มให้อัตโนมัติ** พร้อมแจ้งให้ทราบ
- ถ้าเป็นชื่อใหม่ ระบบจะสร้างรายการสินค้าใหม่ พร้อมให้กำหนดหน่วยและค่าแจ้งเตือน

### นำสินค้าออกจากคลัง
- กดปุ่ม **"นำออก"** ในแถวสินค้าที่ต้องการ
- ใส่จำนวนที่ต้องการนำออก แล้วกด "นำออก" หรือกด Enter
- ระบบจะไม่อนุญาตให้นำออกเกินจำนวนที่มีอยู่

### แจ้งเตือนสินค้าใกล้หมด
- สินค้าแต่ละรายการมีค่า threshold (ค่าแจ้งเตือน) ที่กำหนดได้เอง
- เมื่อจำนวนสินค้าเท่ากับหรือต่ำกว่า threshold จะแสดง badge สีเหลือง **"ใกล้หมด"**
- เมื่อจำนวนเป็น 0 จะแสดง badge สีแดง **"หมด"**
- แถบเตือนด้านบน (header) จะแสดงจำนวนรายการที่ต้องระวังทั้งหมด

### แก้ไขค่าแจ้งเตือน (Threshold)
- ในตารางสินค้า ใต้ชื่อจะแสดง "แจ้งเตือน ≤ X หน่วย"
- คลิกที่ตัวเลข X เพื่อแก้ไขได้ทันที กด Enter หรือคลิกที่อื่นเพื่อบันทึก

### ค้นหาสินค้า
- พิมพ์ในช่อง "ค้นหาสินค้า..." เพื่อกรองรายการแบบ real-time

### ลบสินค้า
- กดปุ่ม **"ลบ"** เพื่อลบรายการสินค้าออกจากคลังถาวร (มีกล่องยืนยันก่อนลบ)

## การบันทึกข้อมูล

ข้อมูลทั้งหมดถูกบันทึกใน **localStorage** ของเบราว์เซอร์ ข้อมูลจะยังอยู่แม้ปิดแท็บหรือรีเฟรชหน้า  
หากล้างข้อมูลเบราว์เซอร์ (clear site data) ข้อมูลจะหายและกลับไปใช้ข้อมูลตัวอย่างเริ่มต้น

---


## ภาษาที่ใช้

- HTML5 / CSS3 / JavaScript (Vanilla) — ไม่มี framework หรือ library ภายนอก
- Google Fonts: IBM Plex Sans Thai, IBM Plex Mono
- Web Storage API (localStorage) สำหรับบันทึกข้อมูล

---

## ข้อจำกัด (Prototype)

- ข้อมูลเก็บใน localStorage ของเครื่องเดียว ไม่ sync ข้ามอุปกรณ์
- ยังไม่มีระบบ login หรือสิทธิ์ผู้ใช้
- ยังไม่มีประวัติการรับ/เบิกสินค้า (transaction log)
- ยังไม่รองรับ export ข้อมูล (CSV / Excel)
